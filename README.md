# Tiago Forman - Barbearia & Estética Masculina

Site de apresentação para o salão Tiago Forman.

## 🚀 Como executar

### Instalação
```bash
npm install
```

### Desenvolvimento
```bash
npm run dev
```

### Build para produção
```bash
npm run build
```

### Preview da build
```bash
npm run preview
```

## 🛠️ Tecnologias

- React 18
- Vite
- Tailwind CSS
- JavaScript/JSX

## 📱 Recursos

- Design moderno e responsivo
- Animações suaves
- Link para Instagram (@thiagoforman)
- Logo personalizada
- Otimizado para performance

## � Deploy no Easypanel

### Configuração
1. Conecte seu repositório GitHub ao Easypanel
2. Crie um novo aplicativo
3. Configure o build:
   - **Build Command**: `npm install && npm run build`
   - **Start Command**: `npx vite preview --host --port $PORT`
   - **Port**: 3000 (ou a porta que o Easypanel atribuir)

### Variáveis de ambiente
Nenhuma variável de ambiente é necessária.
