# Sistema de Agendamento Multi-Tenant - Supabase

Sistema completo de CRM/ERP para salões de beleza e outros negócios com agendamento.

## Estrutura do Projeto

### Backend
- **Supabase** (PostgreSQL + Auth + Storage + Realtime)
- **Row Level Security (RLS)** para multi-tenancy
- **APIs REST/GraphQL** automáticas
- **Triggers e Functions** para automações

### Integrações
- **Evolution API** (WhatsApp)
- **Pagamentos** (preparado para integração)
- **Notificações** (Email + SMS + WhatsApp)

## Características

### Multi-Tenant
- Isolamento completo de dados por organização
- Suporte a múltiplos tipos de negócio
- Configurações personalizáveis por tenant

### Funcionalidades
- ✅ Gestão de Clientes
- ✅ Gestão de Funcionários/Profissionais
- ✅ Catálogo de Serviços
- ✅ Gestão de Produtos
- ✅ Sistema de Agendamento
- ✅ Gestão Financeira
- ✅ Histórico e Auditoria
- ✅ Notificações Automáticas
- ✅ Relatórios e Analytics

## Tecnologias

- **Database**: PostgreSQL (Supabase)
- **Auth**: Supabase Auth
- **Storage**: Supabase Storage
- **Realtime**: Supabase Realtime
- **API**: Auto-gerada pelo Supabase
